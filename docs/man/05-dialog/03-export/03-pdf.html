PDF File Export and Printing

Select this format to create PDF files, or immediately print them.
PDF (Portable Document Format) is a widely used format for high-quality, printable documents.

For example, PDF files will retain the precise page layout when viewed on another system.

A sample PDF file, created with jose, can be viewed here.

Page Setup

Use Page Setup to select the paper size.

Styles

Use Text Styles to customize text styles.

Embed TrueType fonts

The option Embed TrueType fonts will store additional font info in the PDF file.
It is recommended to leave it always checked. If it is not checked, the resulting files are smaller, but they will probably not show up correctly in Acrobat Reader.
If fonts are "embedded", the resulting PDF files can be viewed on any system.

Save As...

Click Save As.. to save the file to disk. Use the File Selection dialog to choose a destination folder and file name.
Use Acrobat Reader to open these files. You can get Acrobat Reader free of charge from Adobe

Print...

Click Print... to print the document to a connected printer.

Preview

Click Preview to view the document on screen before printing it.
Note that previewing PDF documents is quite performance intensive. It might take some time until the previewed page appears.

See The Print Preview Panel .

Caveat: depending on the fonts you are using, the page layout of a printed file will be slightly different from a saved file.
This is a known issue and might be addressed in future versions of jose.
If you need to be sure that the printed version is identical, save the file to disk, then use Acrobat Reader to print it.
