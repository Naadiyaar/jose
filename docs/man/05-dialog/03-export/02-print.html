Printing Games

Select this format to print games to a connected printer. It will print them exactly like they appear in The Game Panel .

Page Setup

Use Page Setup to select the paper size.

Styles

Use Text Styles to customize text styles.

Print...

Click Print... to print to a connected printer.

Preview

Click Preview to view the document on screen. See The Print Preview Panel .

Note that you can also use PDF Print for printing games. PDF usually creates better quality output, but it is also slower.
