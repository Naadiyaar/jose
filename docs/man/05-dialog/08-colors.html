The Color Chooser Dialog

Use this dialog to select colors and textures for Colors , Opening Book Settings and Text Styles .

Choosing a Color

Use the first three tabs to select a color:

  • The Sample tab shows a list of predefined colors.
  • The HSB tab allows you to select HSB (hue-saturation-brightness) color values
  • The RGB tab allows you to select RGB (red-green-blue) color values

Choosing a Texture

Select one from the Texture tab.

Remember to select the Texture tab before clicking OK.

Choosing a Color Gradient

For a color gradient, you need two colors.

  • Click First Color to select the first color.
  • Click Second Color to select the second color.
  • Use the control on the right to determine the direction of the gradient. Click and drag the colored circles.
  • Check cyclic to create a cyclic gradient.

Remember to select the Gradient tab before clicking OK.
