Insert new Move

Choose the operation mode when you insert a new variation ( Variations ).

  • Ask: prompt the user for an option. This is the default setting.
  • New Line: always creates a new variation
  • New Main Line: always creates a new main line. The current main line will become a variation.
  • Overwrite: always overwrites the current line. Use with care!

The last two options are not really useful when applied automatically. If in doubt, choose Ask.
