The Engine Panel

Shows information about the plug-in chess engine. You can play games against the engine, or you can use it for analysing games. See Time Controls .

The color of the leftmost button indicates the current status the the engine:

  • blue button: the engine is shut down. Press the button to start calculating.
  • red button: the engine is actively thinking about its next move
  • green button: the engine is pondering the next move (i.e. it keeps thinking on the opponent's time)
  • yellow button: the engine is doing a background analysis. The results of the analysis will be constantly displayed, but the engine will not make a move on it's own.

Move Now

Start the engine and wait for a move. A second click during the thinking phase will force the engine to move immediately.

Pause

Stop the engine.

Analysis Mode

Start a background analysis. The engine will constantly display its evaluation, but it will not move by itself.
You can use this mode to replay a game and get the engine's evaluation for it.

During the opening phase of a game it will also display moves from the Opening Book. See: Opening Book Settings .
Clicking this button repeatedly alternates between the opening book and the engine analysis.

Get Hint

Request a hint from the engine (see Get Hint ).
The hint will be displayed with a yellow arrow on the board.
Note that the engine is not at all times able to give you a hint.

Status information

Move

Shows the move the engine is currently calculating. This information is only available for UCI engines.

Search Depth

The second field shows the number of plies (half-moves) the engine is thinking in advance.
Examples:

9 / 16
the engine has examined 9 half-moves in advance. Some lines have been examined for 16 half-moves.
BK
the evaluation is taken from the opening book
HT
the evaluation is taken from the engine's internal hash table
TB
the evaluation is taken from the engine's endgame database

Time

Shows the elapsed time for the current calculation.

Node Count

This field shows the number of positions the engine has evaluated so far.

N/sec

Shows the number of positions the engine evalates per second (in average).

Evaluation

The leftmost field shows the current position evaluation, expressed in pawn units.

+3.00
White has an advantage that is worth about 3 pawns
-0.20
Black has an advantage of about 0.2 pawns
+#3
White can mate in three moves
-#4
Black can mate in four moves

Primary Variation

The larger text area at the bottom shows the current primary variation(s), i.e. the best variation the engine has examined so far.

During the opening phase of a game, it will show moves from the opening book. Use the "Analysis" button to alternate between the opening book and the current engine analysis.

Double clicking on line inserts it into the current game, as a new variation.

Context Menu

Move Now

Paste Line
