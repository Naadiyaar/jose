The Help Menu

Context Help

Show help for the current dialog or panel.

Manual

Show the manual main page.

About...

Show the splash screen.

Info...

Show information about jose, its author and contributors.

License...

Show the GPL license.

Home Page

Show the jose home page at http://jose-chess.sourceforge.net. You need to be connected to the Internet, and a Web Browser must be available.
If jose can not find a Web Browser it will ask you to find one.

Online Update

Look for a new release version of jose. If a new release is available, you will be asked to download and install it.
For this to work, you need to be connected to the Internet.

Bug Report

Open a web page for filing bug reports and feature requests.
For this to work, you need to be connected to the Internet.

Select Browser...

Configure the path to your current web browser. Usually, you don't need this option on Windows systems.
