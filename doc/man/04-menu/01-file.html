The File Menu

New

Starts a new game. Already opened games are displayed in separate tabs.

New from here

Starts a new game from the current position.

This is a useful feature if you want to play a series of practice games in a certain opening variation.

New, FRC

Starts a new game of Fischer Random Chess, also known as Full Chess, or Chess 960. The pieces on the base line are set up randomly.

Rules are identical to normal chess, with the exception of castling: the king may be placed on any square of the base line. In a castling move, the king moves to g1 or c1, the associated rook moves to f1 or d1, resp. I.e. the position after castling is the same as in classical chess, but the origin squares of king and rook are usually different.

To perform a castling move, pick up the king and

  • move the king to the square g1, or c1
  • or move the king two squares to the left or right
  • or move the king on top of the associated rook (like you would "capture" it)

Note that in some random positions a castling move may not be discernible from a normal king move. In these rare cases, it is recommended to "capture" the rook, because this is always an unambiguous gesture for castling.

Find out more about Fischer Random Chess at Wikipedia.

Note: Crafty, the chess engine that is installed with jose, can not play Fischer Random Chess properly. You may start a game of FRC against Crafty, but it will not be able to castle.

However, there are many FRC enabled engines around, e.g. Shredder, or Fruit, and some freely available ones, like Spike, The Baron, and many more. Have a look here.

New, Shuffle

Starts a new game of Shuffle Chess. The pieces on the base line are set up randomly.

Castling is only possible if the king and rooks are placed on their traditional squares (a1,e1, and h1). I.e. in most random positions you can't castle at all.

Contrary to FRC, you need not learn special castling rules, and you can play Shuffle Chess against any chess engine, including Crafty.

Open...

Opens a database archive file, or a PGN file. The following file formats are accepted:

  • .jose a Jose archive file
  • .pgn a PGN file
  • .zip a compressed PGN file (using zip, or WinZip)
  • .gz,.gzip a compressed PGN file (using GnuZip)
  • .bz,.bz2 a compressed PGN file (using BZip)
  • .tar a TAR archive (commonly used on Linux, nut rarely for PGN files)

Please note that josï¿½'s concept is a bit different from other chess databases: copying a file into your personal database may take some time, but it need only be done once. After the file has been copied into your database, it will be immediately available in the The Database Overview Panel . You don't have to open the same file again.

Open URL...

Downloads a PGN file from the Internet.

Close

Close the frontmost window. Window Menu

Save

Save the current game into the database. New games are stored in the Autosave folder and can later be moved to other folders.

Save As...

Save the currently selected games to disk. This will open the The Export & Print Dialog , where you can select the target file and various output formats.

The set of selected games depend on the current front window:

  • when the The Game Panel is in front, Save As... applies to all currently opened games.
  • when the The Game List Panel is in front, Save As... applies to the current list of database games.

You can also invoke Save As... from the context menues of The Database Overview Panel and The Game List Panel .

Save All

Save all open games into the database.

Revert Game

Revert to the last saved version of a Game. Note that all modifications are lost!

Print...

Print the currently selected games. This will open the The Export & Print Dialog , where you can select various output formats.

The set of selected games depend on the current front window:

  • when the The Game Panel is in front, Print... applies to all currently opened games.
  • when the The Game List Panel is in front, Print... applies to the current list of database games.

You can also invoke Print... from the context menues of The Database Overview Panel and The Game List Panel .

Page Setup...

Setup paper size for printing. See Page Setup for details.

Quit

Quit the application. If there are modified games, you will be asked to save them into the database.
