The Database Overview Panel

Shows all available Game folders (=colllections).

A new folder is created when you open (import) a PGN file. You are free to create new folders, move games between them, and arrange the folder hierarchy as you like.

Click on a folder to display its contents (the games) in the The Game List Panel . If no folder is selected, all games from all folders will be displayed.

There are three folders with a special purpose:

  • the Auto Save folder stores all games that you play on the The Board Panel (e.g. when you play against Crafty)
  • the Clipboard folder is used as temporary storage when moving or copying games.
  • the Trash folder stores games that are meant to be deleted. Games are not immediately deleted but moved first to the Trash folder. You still have a chance to restore those games. Note however, that once you empty the trash folder, the games are lost forever!

In general, the whole concept should be quite familiar from your Operating System Desktop.

Context Menu

Cut

Cut out this folder, i.e. move it to the clipboard.

Copy

Copy this folder to the clipboard. Note that this can be a lengthy operation for large folders !

Paste

Insert the contents of the clipboard into the selected folder.
When inserting game data from the clipboard, jose will offer two options:

  • Copy: data from the clipboard will be copied into the selected folder. Note that copying large amounts of data can become quite lengthy. If you want to insert the clipboard data just once, it is usually more efficient to:
  • Move games from the clipboard into the selected folder. This option is more appropriate for large amounts of data. However, you can paste the data only once. After this operation, the clipboard is empty again.

Clear

Move this folder to the Trash. It will not be immediately deleted.

Restore

Restore a deleted folder to its original location.

Rename

Rename the folder: edit the folder name and press Enter to commit. Press Escape to abort editing.

New Folder

Creates a new, empty folder.

Move to Top Level

Move this folder to the top level of the hierarchy. You can click a folder and drag it to another folder to modify the hierarchy of folders.

Save As...

Save a collection to disk. This will open the The Export & Print Dialog , where you can select the target file and various output formats.

Print...

Print a set of games. This will open the The Export & Print Dialog , where you can select various output formats.

Classify ECO

Update the opening classification for all games in the folder. See Classify ECO .
