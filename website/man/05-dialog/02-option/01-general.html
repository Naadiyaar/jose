General Settings

Name

Enter your name, as it will appear in the game header (for example, when you play against the "Crafty" engine, the game will be titled "Peter Schäfer - Crafty").

Language

Select the language for menues and dialogs. Currently available are English and German. More languages might be available in the future.

Look & Feel

Select the "Look & Feel" of the application, i.e. the appearance of buttons, menues, etc.
Please note that some Look & Feels are only available on certain platforms (e.g. the "Windows" Look & Feel).

For installing new Look & Feels please refer to ???. The default is "Metouia".

Load Recent Games

Check this option to load all recent games when you start jose.
If this option is not checked, jose will start with a blank game.

Classify Opening by ECO

Automatically calculate the ECO opening codes for the current game (as you play it). Itf not checked, you can calculate the ECO code explicitly with Classify ECO .

Move Announcements:

Select a directory that contains sound files for move announcements. These files are available for various languages.

You can download those files from:

  • jose-chess.sourceforge.net
  • or from: Arena
  • or from: DGT

If you have Fritz installed on your system, you can also use its sound files.
They are usually installed at:
C:\Program Files\ChessBase\Sounds\English\Notation

Announce Engine Moves

If this option is checked, all engine moves will be spoken.

Acknowledge User Moves

If this option is checked, your moves will be acknowledged accoustically (jose will say "OK").

Announce User Moves

If this option is checked, your own moves will be spoken.
