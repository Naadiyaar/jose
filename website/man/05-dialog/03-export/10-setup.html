Page Setup

Use this tab to select a paper format for printing. It is also used when creating PDF files.

Paper

Use the Paper list to select a commonly used paper format.

Size:

You are free to modify the paper size and margins. Use the Size control to edit the width and height of the paper.

Use the popup on the right to select a unit of measurement:

  • mm millimeters
  • cm centimeters
  • in inches (1 inch = 2.54 centimeters = 25.4 millimeters)
  • pt one printer's point (1 pt = 1/72 inch = 0.35 millimeters)

Orientation

Choose the orientation of the page as Portrait or Landscape. These controls are also present in The Print Preview Panel .

Margins

Use the Margins control to set top, left, right and bottom margins.

Click Apply... to apply your settings, or click Revert to revert to the previous settings.
