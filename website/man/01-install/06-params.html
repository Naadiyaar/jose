jose Command Line Options

Setting command line options

Options

datadir=path Path to database director. Default is jose/database, but you might want to store your database at other locations.
splash=off Turn off the splash screen when the program starts.
profile=path path to user profile settings. Default is C:\Windows\Documents\...\.jose.user.preferences
workdir=path path to jose installation directory. Usually, you don't want to change this setting.
db=datas-source Choose a database connection. See config/datasource.xml for details.
